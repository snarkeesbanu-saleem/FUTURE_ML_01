## 📊 Project Overview

This dashboard predicts future **store sales** using historical data from the [Store Sales - Time Series Forecasting](https://www.kaggle.com/competitions/store-sales-time-series-forecasting) dataset.

### Key Features:
- **Interactive Sales Forecasting** (XGBoost + Seasonal Naive)
- **Real-time Filters** (Store, Product Family, Forecast Horizon)
- **Advanced Analytics Tabs**:
  - Holiday Impact Analysis
  - Oil Price Correlation
  - Store Performance (Top Stores)
  - **Feature Importance** (XGBoost)
  - Daily Transactions Analysis
- **Inventory Recommendations** with Safety Stock
- **Downloadable Forecasts** (CSV)
- **Confidence Intervals** on predictions
